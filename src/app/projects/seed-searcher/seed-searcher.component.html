Seed Searcher

Type: Machine Learning Web App

Seed Searcher is a bird identification web application built with Angular for the frontend and C#/ .NET for the backend. It allows users to upload pictures of a bird, and then will identify the bird and give the user information about it. Currently Seed Searcher can identify 325 different species of birds.

SeedSearcher was originally an Android app built with Kotlin for a school project, and could only identify 10 different species. This version has been recreated from scratch for the web and has many more features.
